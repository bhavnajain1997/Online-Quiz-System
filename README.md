# Online Quiz System

## 📌 Project Description
The Online Quiz System is a web-based application that allows users to log in, attempt quizzes, and view their final score. The system is designed to provide a simple and user-friendly interface for conducting online quizzes.

## 🚀 Features
- User Login Authentication
- Multiple Choice Quiz Questions
- Score Calculation
- Result Display

## 🛠️ Technologies Used
- Frontend: React.js
- Backend: Node.js with Express
- Database: MySQL

## 📂 Project Structure
- frontend -> React 
- backend -> Node.js server
- database -> Mysql

## ⚙️ Setup Instructions

1. Install Node.js on your system.

2. Clone the repository:
git clone https://github.com/your-username/online-quiz-system.git

3. Install dependencies:
npm install

4. Create a `.env` file and add the following:
DB_HOST=localhost  
DB_USER=root  
DB_PASSWORD=your_password  
DB_NAME=quiz_db  

5. Start the backend server:
node server.js

6. Run the frontend:
npm start
